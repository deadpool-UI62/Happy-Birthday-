## Features

- **Home Page:**  
  A warm birthday greeting with a beautifully styled main image, smooth fade-in animation, and vibrant navigation.

- **Messages Page:**  
  Displays 7-8 heartwarming birthday wishes revealed one by one with sliding fade animations and a confetti burst effect on each surprise button click.

- **Images Page:**  
  Showcases 5 special photos of Kaarttiiii in a cycling, animated gallery that automatically transitions every 0.5 seconds with smooth scale and fade effects.

- **Cake Animation Page:**  
  Features a realistic 4-layered birthday cake with natural shading and bounce animations, a glowing flickering candle flame, magical sparkles, and an animated vibrant gradient wishing line.

- **Background Audio:**  
  Each page plays a subtle looping background music track that starts muted and unmutes on first user interaction to comply with browser autoplay policies.

## Usage

1. Place your selected birthday music audio file as `background-music.mp3` inside the `audio` folder or update the audio source paths in the HTML files.
2. Put your images of Kaarttiiii in the `images` folder with the naming scheme as above or update image source paths in HTML.
3. Open each HTML file in a modern browser to view the fully animated, interactive birthday website.
4. Customize names, messages, images as desired by editing the respective HTML pages.

---

## Technologies

- HTML5 for structure
- CSS3 with Flexbox, Grid, keyframe animations for stunning visuals and smooth transitions
- Vanilla JavaScript for interactivity such as animation control, confetti, auto image cycling, and background audio management
- Accessibility features including ARIA roles and keyboard-friendly interactions

## Notes

- Background music starts muted for autoplay compatibility, then unmutes softly when the visitor clicks anywhere on the page.
- The layout is responsive and adapts to various device sizes, ensuring an optimal experience across desktops, tablets, and phones.
- All animations are designed to be subtle yet engaging to maintain an elegant celebratory feel.
